A decoration piece with remote and manuelly controllable RGB for our schools First Lego League Team, BBR!!!

# Key Features:
- Can be controlled using IR remote controller and the potentiometers at back.
- Mode change between manuel potentiometer and remote control is done using a switch.
- The IR receiver is placed at front for a clear control.
- Presents a good appearance with the BBR text at the front.
- The blurred plastic covering the text makes the light coming out evenly distributed.
- The holder parts makes the piece stand in a good position.
- A rechargable battery is used.
- The CAD design is made to be manufactured by a CNC carving machine.

  # Wirings
  The wring schematics are prepared using the KiCAD program. It includes all the electrical components used in the project. The pins connected are coherent with the firmware.
